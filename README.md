# clj-grep

Grep kata in Clojure.

In response to:

- https://github.com/shiroyasha/rust_grep
- https://github.com/rastasheep/go-grep
- https://github.com/darkofabijan/elixir-grep

## Usage

```bash
lein run "path/to/directory" "search"
```
